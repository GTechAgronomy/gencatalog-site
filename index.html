Support & FAQ

Video thumbnail setup

Everything you need to know for Intel Mac and Windows on first launch.

Intel Mac — Video Thumbnails

On first launch, GenCatalog may prompt you to install ffmpeg to enable video thumbnail previews. Click "Install via Homebrew" and the installation will run automatically in the background. This is a one-time setup and takes about 1-2 minutes. Image thumbnails work immediately without any additional setup.

Windows — Video Thumbnails

GenCatalog requires ffmpeg to generate video thumbnail previews. On first launch you'll see a prompt with a download link. Download ffmpeg, extract the zip, and add the bin folder to your system PATH. Once installed, restart GenCatalog and video thumbnails will work automatically. Image thumbnails work immediately without any additional setup.
